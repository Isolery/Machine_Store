# 实现对RLM300的数据流储存功能



## 开发套件

+ MCU：ATmega64（8M外部晶振）
+ 时钟芯片：DS1302
+ 存储芯片：AT24C256
+ USB芯片：CH376S
+ 开发平台：Atmel Studio 6.2



## 功能实现

1. 实现iic协议对AT24C256的读取功能
2. 实现DS1302芯片的驱动程序
3. 实现CH376的驱动程序
4. 通过刷卡时对数据流进行解析，提取一帧数据进行存储，其他数据则通过延时关闭串口来舍弃
5. 插入U盘时，自动将存储在256中的新数据拷贝到U盘中
